# 01_2 任务提醒功能

## 功能描述

本功能为任务管理系统添加了任务提醒功能，当任务接近截止时间时，系统会发送系统通知提醒用户，确保用户不会错过重要任务。

## 实现原理

1. **通知工具类**：创建了 `NotificationTool` 类，使用 `plyer` 库发送系统通知
2. **时间检查**：实现了 `is_task_near_deadline` 方法，检查任务是否接近截止时间
3. **后台线程**：使用线程池处理任务检查，避免阻塞主线程
4. **定时检查**：在定时刷新任务列表时检查任务状态，发送提醒

## 使用方法

1. **设置任务**：创建任务时设置截止时间
2. **自动提醒**：当任务接近截止时间（默认5分钟）时，系统会自动发送通知提醒
3. **查看通知**：系统通知会显示任务名称和截止时间

### 核心代码

**通知工具类**：
```python
class NotificationTool:
    @staticmethod
    def send_task_reminder(task_name, deadline):
        """
        发送任务提醒通知
        :param task_name: 任务名称
        :param deadline: 任务截止时间
        :return: bool
        """
        try:
            notification.notify(
                title="任务提醒",
                message=f"任务 '{task_name}' 即将到期，截止时间：{deadline}",
                app_name="Task Management",
                timeout=10  # 通知显示10秒
            )
            logger.info(f"发送任务提醒通知成功：{task_name}")
            return True
        except Exception as e:
            logger.error(f"发送任务提醒通知失败：{e}")
            return False
```

**时间检查方法**：
```python
@staticmethod
def is_task_near_deadline(end_time, minutes=5):
    """
    检查任务是否接近截止时间
    :param end_time: 任务截止时间
    :param minutes: 提前提醒的分钟数
    :return: bool
    """
    current_time = TimeProcess.get_current_time()
    time_diff = TimeProcess.calculate_time_poor(current_time, end_time)
    return 0 < time_diff <= minutes * 60
```

**任务检查线程**：
```python
class TaskChecker(QRunnable):
    """任务检查线程"""
    def __init__(self, main_window):
        super().__init__()
        self.main_window = main_window
        self.signals = TaskCheckerSignals()
        
    def run(self):
        """运行任务检查"""
        try:
            # 检查任务超时和接近截止时间
            for task in self.main_window.unfinished_list_task_data:
                end_time = task.get("end_time", None)
                if not end_time:
                    continue
                
                # 检查是否接近截止时间
                if TimeProcess.is_task_near_deadline(end_time):
                    task_name = task.get("name", "未命名任务")
                    NotificationTool.send_task_reminder(task_name, end_time)
        except Exception as e:
            logger.error(f"任务检查线程出错：{e}")
```

## 效果展示

- 当任务接近截止时间时，系统会发送通知
- 通知包含任务名称和截止时间
- 通知会在系统通知栏显示，持续10秒

## 注意事项

- 提醒时间默认为5分钟，可以根据需要调整
- 通知功能依赖于 `plyer` 库，请确保已安装
- 系统需要允许应用发送通知
