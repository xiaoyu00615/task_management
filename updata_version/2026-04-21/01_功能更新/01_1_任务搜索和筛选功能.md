# 01_1 任务搜索和筛选功能

## 功能描述

本功能为任务管理页面添加了搜索和筛选功能，允许用户通过任务名称搜索和按分类筛选任务，提高任务管理的效率。

## 实现原理

1. **UI 组件添加**：在任务页面顶部添加了搜索框和筛选下拉框
2. **事件绑定**：为搜索框和筛选下拉框绑定了文本变化和选择变化事件
3. **过滤逻辑**：实现了 `filter_tasks` 方法，根据搜索文本和筛选条件过滤任务
4. **列表更新**：过滤后重新加载任务列表，实时显示符合条件的任务

## 使用方法

1. **搜索任务**：在搜索框中输入任务名称，系统会实时过滤显示包含该文本的任务
2. **筛选任务**：从筛选下拉框中选择分类，系统会显示该分类下的所有任务
3. **组合使用**：可以同时使用搜索和筛选功能，系统会同时应用两个条件

**过滤逻辑**：
```python
def filter_tasks(self, search_text=None, filter_category=None):
    """筛选任务
    
    Args:
        search_text: 搜索文本
        filter_category: 筛选分类
    """
    if not self.original_list:
        return
    
    # 重置为原始列表
    filtered_tasks = self.original_list.copy()
    
    # 应用搜索过滤
    if search_text:
        search_text = search_text.lower()
        filtered_tasks = [task for task in filtered_tasks if search_text in task.get('name', '').lower()]
    
    # 应用分类过滤
    if filter_category:
        filtered_tasks = [task for task in filtered_tasks if task.get('category') == filter_category]
    
    # 更新列表并重新加载
    self.choose_list = filtered_tasks
    self.del_all_tasks_ui()
    self.label_value.setText(f"{self.text_map.get(self.status,None)}:({len(self.choose_list)})")
    self._load_items()
```

- 筛选下拉框：选择分类，显示该分类下的任务
- 组合使用：同时应用搜索和筛选条件，显示符合条件的任务

## 注意事项

- 搜索区分大小写
- 筛选基于任务的分类字段
- 当搜索框为空且筛选为"全部"时，显示所有任务
