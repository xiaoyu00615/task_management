# 更新总览

## 更新日期

2026-04-21

## 更新内容

### 01\_功能更新

#### 01\_1 任务搜索和筛选功能

- **功能描述**：为任务管理页面添加了搜索和筛选功能，允许用户通过任务名称搜索和按分类筛选任务
- **使用方法**：在任务页面的搜索框中输入任务名称，或从筛选下拉框中选择分类
- **实现原理**：添加了搜索框和筛选下拉框，实现了过滤逻辑，实时更新任务列表

#### 01\_2 任务提醒功能

- **功能描述**：当任务接近截止时间时，系统会发送系统通知提醒用户
- **使用方法**：创建任务时设置截止时间，系统会自动在任务接近截止时间时发送提醒
- **实现原理**：使用 `plyer` 库发送系统通知，在后台线程中检查任务状态

### 02\_代码优化

#### 02\_1 代码结构优化

- **优化内容**：为核心模块添加了详细的文档字符串，整理了模块结构，优化了导入方式
- **实现原理**：添加了 `__all__` 导出列表，使用相对导入和统一的导出机制
- **效果**：代码结构更加清晰，可维护性和可读性提高

#### 02\_2 性能优化

- **优化内容**：对 JSON 文件读写操作进行了性能优化，添加了缓存机制
- **实现原理**：为 `FileProcess` 类添加了缓存机制，减少文件 I/O 操作，实现了缓存过期机制
- **效果**：读写效率显著提高，减少了重复的文件操作

#### 02\_3 用户体验优化

- **优化内容**：调整了 UI 布局，提高了响应速度
- **实现原理**：添加了颜色映射缓存，使用线程池处理耗时操作，避免阻塞主线程
- **效果**：界面更加美观，操作更加流畅

### 03\_安全性增强

#### 03\_1 日记加密机制加强

- **优化内容**：添加了 AES-256 加密支持，提高了日记数据的安全性
- **实现原理**：使用 `cryptography` 库实现 AES-256 加密，使用 SHA-256 生成密钥，实现了回退机制
- **效果**：日记数据安全性显著提高，同时保持了兼容性

## 安装和使用

### 安装依赖

```bash
# 创建虚拟环境
python -m venv venv

# 激活虚拟环境
venv\Scripts\activate

# 安装依赖
pip install -r requirements.txt
```

### 运行应用

```bash
# 运行应用
python main.py
```

## 注意事项

1. **任务搜索和筛选**：搜索区分大小写，筛选基于任务的分类字段
2. **任务提醒**：提醒时间默认为5分钟，需要系统允许应用发送通知
3. **性能优化**：缓存默认过期时间为5分钟，可以根据需要调整
4. **安全性**：需要安装 `cryptography` 库以支持 AES 加密
5. **兼容性**：加密失败时会自动回退到简单加密，确保兼容性
