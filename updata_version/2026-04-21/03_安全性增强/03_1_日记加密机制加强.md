# 03_1 日记加密机制加强

## 优化内容

本优化对项目的日记加密机制进行了加强，添加了 AES-256 加密支持，提高了日记数据的安全性。

## 实现原理

1. **AES 加密**：使用 `cryptography` 库实现 AES-256 加密，提供更高级别的安全性
2. **密钥处理**：使用 SHA-256 生成加密密钥，确保密钥的安全性
3. **回退机制**：实现了加密失败时的回退机制，确保兼容性
4. **初始化向量**：使用随机 IV（初始化向量），提高加密的安全性

### 核心代码

**AES 加密实现**：
```python
def encrypt(self, plaintext):
    """
    加密方法：明文 → 密文（使用 AES 加密）
    :param plaintext: 明文
    :return: str 加密后的密文（base64 编码）
    """
    if not plaintext:
        return ""

    try:
        # 生成随机 IV（初始化向量）
        iv = os.urandom(16)
        # 创建 AES 加密器
        cipher = Cipher(
            algorithms.AES(self.aes_key),
            modes.CBC(iv),
            backend=default_backend()
        )
        encryptor = cipher.encryptor()
        
        # 对数据进行填充
        padder = padding.PKCS7(128).padder()
        padded_data = padder.update(plaintext.encode()) + padder.finalize()
        
        # 加密数据
        ciphertext = encryptor.update(padded_data) + encryptor.finalize()
        
        # 将 IV 和密文组合并 base64 编码
        combined = iv + ciphertext
        return base64.b64encode(combined).decode('utf-8')
    except Exception as e:
        # 如果 AES 加密失败，回退到简单加密
        return self._simple_encrypt(plaintext)
```

**AES 解密实现**：
```python
def decrypt(self, ciphertext):
    """
    解密方法：密文 → 明文（使用 AES 解密）
    :param ciphertext: 加密后的密文（base64 编码）
    :return: str 解密后的明文
    """
    if not ciphertext:
        return ""

    try:
        # 解码 base64
        combined = base64.b64decode(ciphertext.encode('utf-8'))
        # 提取 IV 和密文
        iv = combined[:16]
        encrypted_data = combined[16:]
        
        # 创建 AES 解密器
        cipher = Cipher(
            algorithms.AES(self.aes_key),
            modes.CBC(iv),
            backend=default_backend()
        )
        decryptor = cipher.decryptor()
        
        # 解密数据
        padded_plaintext = decryptor.update(encrypted_data) + decryptor.finalize()
        
        # 移除填充
        unpadder = padding.PKCS7(128).unpadder()
        plaintext = unpadder.update(padded_plaintext) + unpadder.finalize()
        
        return plaintext.decode('utf-8')
    except Exception as e:
        # 如果 AES 解密失败，回退到简单解密
        return self._simple_decrypt(ciphertext)
```

**密钥生成**：
```python
def _generate_aes_key(self, key):
    """生成 AES 密钥"""
    # 使用 SHA-256 生成 32 字节的密钥
    import hashlib
    return hashlib.sha256(key.encode()).digest()
```

## 效果展示

- 日记数据使用 AES-256 加密，安全性显著提高
- 加密失败时会自动回退到简单加密，确保兼容性
- 密钥使用 SHA-256 生成，更加安全

## 注意事项

- 需要安装 `cryptography` 库：`pip install cryptography==43.0.0`
- AES 加密使用 base64 编码，密文长度会比明文长
- 随机 IV 确保了相同明文加密后的密文不同，提高了安全性
